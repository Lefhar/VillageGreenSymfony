◆ isPostInsertGenerator()

isPostInsertGenerator ( )

Gets whether this generator is a post-insert generator which means that generateId() must be called after the entity has been inserted into the database.

By default, this method returns FALSE. Generators that have this requirement must override this method and return TRUE.
