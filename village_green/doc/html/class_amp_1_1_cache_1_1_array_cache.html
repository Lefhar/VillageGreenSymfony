delete ( string  $key)

Deletes a value associated with the given key if it exists.Implementations SHOULD return boolean true or false to indicate whether the specified key existed at the time the delete operation was requested. If such information is not available, the implementation MUST resolve the promise with null.Implementations MUST transparently succeed operations for non-existent keys.

Parameters
$keystring Cache key.
Returns
Promise<bool|null> Resolves to true / false to indicate whether the key existed or fails with a CacheException on failure. May also resolve with null if that information is not available.
