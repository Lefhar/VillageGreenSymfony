◆ printFormatPreserving()

printFormatPreserving ( array  $stmts,
array  $origStmts,
array  $origTokens 
)

Perform a format-preserving pretty print of an AST.

The format preservation is best effort. For some changes to the AST the formatting will not be preserved (at least not locally).

In order to use this method a number of prerequisites must be satisfied:

  • The startTokenPos and endTokenPos attributes in the lexer must be enabled.
  • The CloningVisitor must be run on the AST prior to modification.
  • The original tokens must be provided, using the getTokens() method on the lexer.
Parameters
Node[]$stmtsModified AST with links to original AST
Node[]$origStmtsOriginal AST with token offset information
array$origTokensTokens of the original code
Returns
string
