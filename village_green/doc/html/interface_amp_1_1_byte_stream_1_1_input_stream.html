 read ()
 

Detailed Description

An InputStream allows reading byte streams in chunks.

Example

function readAll(InputStream $in): Promise {
return Amp\call(function () use ($in) {
$buffer = "";
while (($chunk = yield $in->read()) !== null) {
$buffer .= $chunk;
}
return $buffer;
});
}
