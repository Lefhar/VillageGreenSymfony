Detailed Description

Creates a buffered message from an InputStream. The message can be consumed in chunks using the read() API or it may be buffered and accessed in its entirety by calling buffer(). Once buffering is requested through buffer(), the stream cannot be read in chunks. On destruct any remaining data is read from the InputStream given to this class.
