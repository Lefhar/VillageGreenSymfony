Intercepts an HTTP request after the connection to the remote server has been established.

The implementation might modify the request and/or modify the response after the promise returned from {\ttfamily \$stream-\/\texorpdfstring{$>$}{>}request(...)} resolved.

A \mbox{\hyperlink{interface_amp_1_1_http_1_1_client_1_1_network_interceptor}{Network\+Interceptor}} SHOULD NOT short-\/circuit and SHOULD delegate to the {\ttfamily \$stream} passed as third argument exactly once. The only exception to this is throwing an exception, e.\+g. because the TLS settings used are unacceptable. If you need short circuits, use an \{\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{interface_amp_1_1_http_1_1_client_1_1_application_interceptor}{Application\+Interceptor}}\} instead.
