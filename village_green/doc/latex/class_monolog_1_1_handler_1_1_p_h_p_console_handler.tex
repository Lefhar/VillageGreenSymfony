\doxysubsubsection{\texorpdfstring{handle()}{handle()}}
{\footnotesize\ttfamily handle (\begin{DoxyParamCaption}\item[{array}]{\$record }\end{DoxyParamCaption})}

Handles a record.

All records may be passed to this method, and the handler should discard those that it does not want to handle.

The return value of this function controls the bubbling process of the handler stack. Unless the bubbling is interrupted (by returning true), the \mbox{\hyperlink{class_monolog_1_1_logger}{Logger}} class will keep on calling further handlers in the stack with a given log record.

\end{DoxyParams}
\begin{DoxyReturn}{Returns}
bool true means that this handler handled the record, and that bubbling is not permitted. false means the record was either not processed or that this handler allows bubbling.
