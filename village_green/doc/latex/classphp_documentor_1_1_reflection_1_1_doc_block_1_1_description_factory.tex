\doxysubsection{Detailed Description}
Creates a new \mbox{\hyperlink{classphp_documentor_1_1_reflection_1_1_doc_block_1_1_description}{Description}} object given a body of text.

Descriptions in php\+Documentor are somewhat complex entities as they can contain one or more tags inside their body that can be replaced with a readable output. The replacing is done by passing a Formatter object to the \mbox{\hyperlink{classphp_documentor_1_1_reflection_1_1_doc_block_1_1_description}{Description}} object\textquotesingle{}s {\ttfamily render} method.

In addition to the above does a \mbox{\hyperlink{classphp_documentor_1_1_reflection_1_1_doc_block_1_1_description}{Description}} support two types of escape sequences\+:


\begin{DoxyEnumerate}
\item {\ttfamily \{@\}} to escape the {\ttfamily @} character to prevent it from being interpreted as part of a tag, i.\+e. {\ttfamily \{\{@\}link\}}
\item {\ttfamily \{\}} to escape the {\ttfamily \}} character, this can be used if you want to use the {\ttfamily \}} character in the description of an inline tag.
\end{DoxyEnumerate}

If a body consists of multiple lines then this factory will also remove any superfluous whitespace at the beginning of each line while maintaining any indentation that is used. This will prevent formatting parsers from tripping over unexpected spaces as can be observed with tag descriptions.
