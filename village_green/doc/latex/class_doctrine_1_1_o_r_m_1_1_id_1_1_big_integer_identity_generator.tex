\doxysubsection{Detailed Description}
Id generator that obtains IDs from special \char`\"{}identity\char`\"{} columns. These are columns that automatically get a database-\/generated, auto-\/incremented identifier on INSERT. This generator obtains the last insert id after such an insert.
