\doxysubsection{Detailed Description}
Manages \mbox{\hyperlink{class_doctrine_1_1_deprecations_1_1_deprecation}{Deprecation}} logging in different ways.

By default triggered exceptions are not logged.

To enable different deprecation logging mechanisms you can call the following methods\+:


\begin{DoxyItemize}
\item Minimal collection of deprecations via \mbox{\hyperlink{class_doctrine_1_1_deprecations_1_1_deprecation_afc998b295c40d8db5a6a7bfff00048e3}{get\+Triggered\+Deprecations()}} \textbackslash{}\+Doctrine\textbackslash{}\+Deprecations\textbackslash{}\+Deprecation\+::enable\+Tracking\+Deprecations();
\item Uses @trigger\+\_\+error with E\+\_\+\+USER\+\_\+\+DEPRECATED \textbackslash{}\+Doctrine\textbackslash{}\+Deprecations\textbackslash{}\+Deprecation\+::enable\+With\+Trigger\+Error();
\item Sends deprecation messages via a PSR-\/3 logger \textbackslash{}\+Doctrine\textbackslash{}\+Deprecations\textbackslash{}\+Deprecation\+::enable\+With\+Psr\+Logger(\$logger);
\end{DoxyItemize}

Packages that trigger deprecations should use the {\ttfamily \mbox{\hyperlink{class_doctrine_1_1_deprecations_1_1_deprecation_a1f761f43956fd80d86129ca1a86ed897}{trigger()}}} or {\ttfamily \mbox{\hyperlink{class_doctrine_1_1_deprecations_1_1_deprecation_a213bd7ec4e6e5468b9b39eeafab493b4}{trigger\+If\+Called\+From\+Outside()}}} methods. \begin{Desc}

Trigger a deprecation for the given package and identifier when called from outside.

\char`\"{}\+Outside\char`\"{} means we assume that \$package is currently installed as a dependency and the caller is not a file in that package. When \$package is installed as a root package then deprecations triggered from the tests folder are also considered \char`\"{}outside\char`\"{}.

This deprecation method assumes that you are using Composer to install the dependency and are using the default /vendor/ folder and not a Composer plugin to change the install location. The assumption is also that \$package is the exact composer packge name.

Compared to \mbox{\hyperlink{class_doctrine_1_1_deprecations_1_1_deprecation_a1f761f43956fd80d86129ca1a86ed897}{trigger()}} this method causes some overhead when deprecation tracking is enabled even during deduplication, because it needs to call \mbox{\hyperlink{}{debug\+\_\+backtrace()}}
