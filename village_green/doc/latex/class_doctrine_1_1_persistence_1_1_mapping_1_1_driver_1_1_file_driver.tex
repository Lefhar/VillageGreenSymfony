\doxysubsection{Detailed Description}
Base driver for file-\/based metadata drivers.

A file driver operates in a mode where it loads the mapping files of individual classes on demand. This requires the user to adhere to the convention of 1 mapping file per class and the file names of the mapping files must correspond to the full class name, including namespace, with the namespace delimiters \textquotesingle{}\textbackslash{}\textquotesingle{}, replaced by dots \textquotesingle{}.\textquotesingle{}.
