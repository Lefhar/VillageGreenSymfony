Concatenate two names, yielding a new \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name}{Name}} instance.

The type of the generated instance depends on which class this method is called on, for example \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name}{Name}}\textbackslash{}\+Fully\+Qualified\+::concat() will yield a \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name}{Name}}\textbackslash{}\+Fully\+Qualified instance.

If one of the arguments is null, a new instance of the other name will be returned. If both arguments are null, null will be returned. As such, writing \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name_a22ea554118d789f15e027ff685016b47}{Name\+::concat}}(\$namespace, \$short\+Name) where \$namespace is a \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name}{Name}} node or null will work as expected.

Gets a slice of a name (similar to array\+\_\+slice).

This method returns a new instance of the same type as the original and with the same attributes.

If the slice is empty, null is returned. The null value will be correctly handled in concatenations using \mbox{\hyperlink{class_php_parser_1_1_node_1_1_name_a22ea554118d789f15e027ff685016b47}{concat()}}.

Offset and length have the same meaning as in array\+\_\+slice().
