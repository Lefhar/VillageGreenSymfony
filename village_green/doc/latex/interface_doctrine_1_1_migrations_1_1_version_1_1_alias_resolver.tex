{\footnotesize\ttfamily resolve\+Version\+Alias (\begin{DoxyParamCaption}\item[{string}]{\$alias }\end{DoxyParamCaption})}

Returns the version number from an alias.

Supported aliases are\+:


\begin{DoxyItemize}
\item first\+: The very first version before any migrations have been run.
\item current\+: The current version.
\item prev\+: The version prior to the current version.
\item next\+: The version following the current version.
\item latest\+: The latest available version.
\end{DoxyItemize}

If an existing version number is specified, it is returned verbatim.
