\doxysubsection{Detailed Description}
Variable Date\+Time \mbox{\hyperlink{class_doctrine_1_1_d_b_a_l_1_1_types_1_1_type}{Type}} using date\+\_\+create() instead of Date\+Time\+::create\+From\+Format().

This type has performance implications as it runs twice as long as the regular \{\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_doctrine_1_1_d_b_a_l_1_1_types_1_1_date_time_type}{Date\+Time\+Type}}\}, however in certain Postgre\+SQL configurations with TIMESTAMP(n) columns where n \texorpdfstring{$>$}{>} 0 it is necessary to use this type.
