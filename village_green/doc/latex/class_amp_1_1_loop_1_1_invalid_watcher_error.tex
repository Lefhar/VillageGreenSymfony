\doxysubsection{Detailed Description}
MUST be thrown if any operation (except \mbox{\hyperlink{class_amp_1_1_loop_ae14f859352356574fc168da4f791a4c1}{disable()}} and \mbox{\hyperlink{class_amp_1_1_loop_aafb9917ec3d532e2ef42546f72aca27f}{cancel()}}) is attempted with an invalid watcher identifier.

An invalid watcher identifier is any identifier that is not yet emitted by the driver or cancelled by the user.
