\doxysubsection{Detailed Description}
A {\ttfamily \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader}{Class\+Loader}}} is an autoloader for class files that can be installed on the SPL autoload stack. It is a class loader that either loads only classes of a specific namespace or all namespaces and it is suitable for working together with other autoloaders in the SPL autoload stack.

If no include path is configured through the constructor or \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader_a24c05c7b3a2132d294fe918e1e64c416}{set\+Include\+Path}}, a \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader}{Class\+Loader}} relies on the PHP {\ttfamily include\+\_\+path}.

\item[\mbox{\hyperlink{deprecated__deprecated000039}{Deprecated}}]The \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader}{Class\+Loader}} is deprecated and will be removed in version 4.\+0 of doctrine/common. \end{DoxyRefDesc}

\doxysubsubsection{\texorpdfstring{classExists()}{classExists()}}
{\footnotesize\ttfamily static class\+Exists (\begin{DoxyParamCaption}\item[{}]{\$class\+Name }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Checks whether a class with a given name exists. A class \char`\"{}exists\char`\"{} if it is either already defined in the current request or if there is an autoloader on the SPL autoload stack that is a) responsible for the class in question and b) is able to load a class file in which the class definition resides.

If the class is not already defined, each autoloader in the SPL autoload stack is asked whether it is able to tell if the class exists. If the autoloader is a {\ttfamily \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader}{Class\+Loader}}}, \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader_a7e2e1700a69d975ff0fc491120f5eb49}{can\+Load\+Class}} is used, otherwise the autoload function of the autoloader is invoked and expected to return a value that evaluates to TRUE if the class (file) exists. As soon as one autoloader reports that the class exists, TRUE is returned.

Note that, depending on what kinds of autoloaders are installed on the SPL autoload stack, the class (file) might already be loaded as a result of checking for its existence. This is not the case with a {\ttfamily \mbox{\hyperlink{class_doctrine_1_1_common_1_1_class_loader}{Class\+Loader}}}, who separates these responsibilities.
