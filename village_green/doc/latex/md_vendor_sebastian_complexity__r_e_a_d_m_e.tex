\href{https://packagist.org/packages/sebastian/complexity}{\texttt{ }} \href{https://php.net/}{\texttt{ }} \href{https://phpunit.de/build-status.html}{\texttt{ }} \href{https://shepherd.dev/github/sebastianbergmann/complexity}{\texttt{ }}\hypertarget{md_vendor_sebastian_complexity__r_e_a_d_m_e_autotoc_md1069}{}\doxysection{Installation}\label{md_vendor_sebastian_complexity__r_e_a_d_m_e_autotoc_md1069}
You can add this library as a local, per-\/project dependency to your project using \href{https://getcomposer.org/}{\texttt{ Composer}}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{composer require sebastian/complexity}

\end{DoxyCode}


If you only need this library during development, for instance to run your project\textquotesingle{}s test suite, then you should add it as a development-\/time dependency\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{composer require -\/-\/dev sebastian/complexity}

\end{DoxyCode}
