\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_amp_1_1_emitter}{Emitter}} is a container for an iterator that can emit values using the \mbox{\hyperlink{class_amp_1_1_emitter_a0060b191dbb4ce03b77fded4d66860b8}{emit()}} method and completed using the \mbox{\hyperlink{class_amp_1_1_emitter_a97c73acf76965d89f77bcc5d6d158d22}{complete()}} and \mbox{\hyperlink{class_amp_1_1_emitter_a5df97dc03c3c97cdf40b90fbf101f54f}{fail()}} methods of this object. The contained iterator may be accessed using the \mbox{\hyperlink{class_amp_1_1_emitter_ad39c70322389b8d8c48a65e41b5a93c7}{iterate()}} method. This object should not be part of a public API, but used internally to create and emit values to an iterator.
