\doxysubsection{Detailed Description}
Represents a collection type as described in the PSR-\/5, the PHPDoc Standard.

A collection can be represented in two forms\+:


\begin{DoxyEnumerate}
\item {\ttfamily ACollection\+Object\texorpdfstring{$<$}{<}a\+Value\+Type\texorpdfstring{$>$}{>}}
\item {\ttfamily ACollection\+Object\texorpdfstring{$<$}{<}a\+Value\+Type,a\+Key\+Type\texorpdfstring{$>$}{>}}
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item ACollection\+Object can be \textquotesingle{}array\textquotesingle{} or an object that can act as an array
\item a\+Value\+Type and a\+Key\+Type can be any type expression
\end{DoxyItemize}
