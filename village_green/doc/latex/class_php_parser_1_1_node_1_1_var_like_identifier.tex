\doxysubsection{Detailed Description}
Represents a name that is written in source code with a leading dollar, but is not a proper variable. The leading dollar is not stored as part of the name.

\mbox{\hyperlink{namespace_examples}{Examples}}\+: Names in property declarations are formatted as variables. Names in static property lookups are also formatted as variables.
