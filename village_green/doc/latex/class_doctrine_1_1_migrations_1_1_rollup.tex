\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_rollup}{Rollup}} class is responsible for deleting all previously executed migrations from the versions table and marking the freshly dumped schema migration (that was created with \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_schema_dumper}{Schema\+Dumper}}) as migrated.
