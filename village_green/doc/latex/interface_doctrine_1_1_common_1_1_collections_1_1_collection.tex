\doxysubsection{Detailed Description}
The missing (SPL) Collection/\+Array/\+Ordered\+Map interface.

A \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_collections_1_1_collection}{Collection}} resembles the nature of a regular PHP array. That is, it is essentially an {\bfseries{ordered map}} that can also be used like a list.

A \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_collections_1_1_collection}{Collection}} has an internal iterator just like a PHP array. In addition, a \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_collections_1_1_collection}{Collection}} can be iterated with external iterators, which is preferable. To use an external iterator simply use the foreach language construct to iterate over the collection (which calls \mbox{\hyperlink{}{get\+Iterator()}} internally) or explicitly retrieve an iterator though \mbox{\hyperlink{}{get\+Iterator()}} which can then be used to iterate over the collection. You can not rely on the internal iterator of the collection being at a certain position unless you explicitly positioned it before. Prefer iteration with external iterators.
