You can add this library as a local, per-\/project dependency to your project using \href{https://getcomposer.org/}{\texttt{ Composer}}\+: \begin{DoxyVerb}composer require phpunit/php-file-iterator
\end{DoxyVerb}

\end{DoxyVerb}
