\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_abstract_migration}{Abstract\+Migration}} class is for end users to extend from when creating migrations. Extend this class and implement the required \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_abstract_migration_a23eb1c1428e8ea2ab2cf798fc06ec421}{up()}} and \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_abstract_migration_aa8eb70255a46429d4d6165c778c9e5b9}{down()}} methods.
