Truths which we believe to be self-\/evident\+:


\begin{DoxyItemize}
\item {\bfseries{It\textquotesingle{}s an asynchronous world.}} Be wary of anything that undermines async principles.
\item {\bfseries{The answer is not more options.}} If you feel compelled to expose new preferences to the user it\textquotesingle{}s very possible you\textquotesingle{}ve made a wrong turn somewhere.
\item {\bfseries{There are no power users.}} The idea that some users \char`\"{}understand\char`\"{} concepts better than others has proven to be, for the most part, false. If anything, \char`\"{}power users\char`\"{} are more dangerous than the rest, and we should avoid exposing dangerous functionality to them.
