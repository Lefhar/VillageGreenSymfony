Retrieves cached information from the data store.

The server\textquotesingle{}s statistics array has the following values\+:


\begin{DoxyItemize}
\item {\bfseries{hits}} Number of keys that have been requested and found present.
\item {\bfseries{misses}} Number of items that have been requested and not found.
\item {\bfseries{uptime}} Time that the server is running.
\item {\bfseries{memory\+\_\+usage}} Memory used by this server to store items.
\item {\bfseries{memory\+\_\+available}} Memory allowed to use for storage.
\end{DoxyItemize}
