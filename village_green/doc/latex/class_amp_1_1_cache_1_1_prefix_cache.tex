{\footnotesize\ttfamily delete (\begin{DoxyParamCaption}\item[{string}]{\$key }\end{DoxyParamCaption})}

Deletes a value associated with the given key if it exists.

Implementations SHOULD return boolean {\ttfamily true} or {\ttfamily false} to indicate whether the specified key existed at the time the delete operation was requested. If such information is not available, the implementation MUST resolve the promise with {\ttfamily null}.

Implementations MUST transparently succeed operations for non-\/existent keys.
