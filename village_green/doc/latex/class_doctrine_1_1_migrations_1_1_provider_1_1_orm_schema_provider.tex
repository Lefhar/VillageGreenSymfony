\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_provider_1_1_orm_schema_provider}{Orm\+Schema\+Provider}} class is responsible for creating a Doctrine\textbackslash{}\+DBAL\textbackslash{}\+Schema\textbackslash{}\+Schema instance from the mapping information provided by the Doctrine ORM. This is then used to diff against your current database schema to produce a migration to bring your database in sync with the ORM mapping information.
