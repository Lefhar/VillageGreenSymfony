\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister}{Basic\+Entity\+Persister}} maps an entity to a single table in a relational database.

A persister is always responsible for a single entity type.

Entity\+Persisters are used during a \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_unit_of_work}{Unit\+Of\+Work}} to apply any changes to the persistent state of entities onto a relational database when the \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_unit_of_work}{Unit\+Of\+Work}} is committed, as well as for basic querying of entities and their associations (not DQL).

The persisting operations that are invoked during a commit of a \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_unit_of_work}{Unit\+Of\+Work}} to persist the persistent entity state are\+:


\begin{DoxyItemize}
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a83e707f7e04878e6b7e1c9e7f9bd9c9e}{add\+Insert}} \+: To schedule an entity for insertion.
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_add6709b93c551f3441499dbfe90925ef}{execute\+Inserts}} \+: To execute all scheduled insertions.
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a73a92552282d02a30c3e1b6ec7835ec7}{update}} \+: To update the persistent state of an entity.
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a4e8fafc58c59edd43fb68f9912f85f0c}{delete}} \+: To delete the persistent state of an entity.
\end{DoxyItemize}

As can be seen from the above list, insertions are batched and executed all at once for increased efficiency.

The querying operations invoked during a \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_unit_of_work}{Unit\+Of\+Work}}, either through direct find requests or lazy-\/loading, are the following\+:


\begin{DoxyItemize}
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a398cf0c1b5d1b9d9f3c2fef2dd4f72cd}{load}} \+: Loads (the state of) a single, managed entity.
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a019c0156bbe2119fad340d5f23cbb0f3}{load\+All}} \+: Loads multiple, managed entities.
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_a9c1368432fcda4fec435d5413138cc1b}{load\+One\+To\+One\+Entity}} \+: Loads a one/many-\/to-\/one entity association (lazy-\/loading).
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_af98f126b6b6328275db2b00d4096e0e5}{load\+One\+To\+Many\+Collection}} \+: Loads a one-\/to-\/many entity association (lazy-\/loading).
\item \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister_ae801738bbd5420b9e75df10eec523c0e}{load\+Many\+To\+Many\+Collection}} \+: Loads a many-\/to-\/many entity association (lazy-\/loading).
\end{DoxyItemize}

The \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_persisters_1_1_entity_1_1_basic_entity_persister}{Basic\+Entity\+Persister}} implementation provides the default behavior for persisting and querying entities that are mapped to a single database table.

Subclasses can be created to provide custom persisting and querying strategies, i.\+e. spanning multiple tables.
