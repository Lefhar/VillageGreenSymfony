\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_doctrine_1_1_common_1_1_collections_1_1_array_collection}{Array\+Collection}} is a \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_collections_1_1_collection}{Collection}} implementation that wraps a regular PHP array.

Warning\+: Using (un-\/)serialize() on a collection is not a supported use-\/case and may break when we change the internals in the future.
