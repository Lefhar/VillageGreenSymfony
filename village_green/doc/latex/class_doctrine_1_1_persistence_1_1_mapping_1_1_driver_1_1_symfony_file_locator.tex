\doxysubsection{Detailed Description}
The Symfony File Locator makes a simplifying assumptions compared to the \mbox{\hyperlink{class_doctrine_1_1_persistence_1_1_mapping_1_1_driver_1_1_default_file_locator}{Default\+File\+Locator}}. By assuming paths only contain entities of a certain namespace the mapping files consists of the short classname only.
