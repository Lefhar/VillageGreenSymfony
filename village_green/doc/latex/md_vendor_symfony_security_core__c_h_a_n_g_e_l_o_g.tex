\begin{DoxyItemize}
\item Add a {\ttfamily Cacheable\+Voter\+Interface} for voters that vote only on identified attributes and subjects
\item Deprecate {\ttfamily Authentication\+Events\+::\+AUTHENTICATION\+\_\+\+FAILURE}, use the {\ttfamily Login\+Failure\+Event} instead
\item Deprecate {\ttfamily Anonymous\+Token}, as the related authenticator was deprecated in 5.\+3
\item Deprecate {\ttfamily Token\+::get\+Credentials()}, tokens should no longer contain credentials (as they represent authenticated sessions)
\item Deprecate returning {\ttfamily string$\vert$\textbackslash{}Stringable} from {\ttfamily Token\+::get\+User()} (it must return a {\ttfamily User\+Interface})
\item Deprecate {\ttfamily Authenticated\+Voter\+::\+IS\+\_\+\+AUTHENTICATED\+\_\+\+ANONYMOUSLY} and {\ttfamily Authenticated\+Voter\+::\+IS\+\_\+\+ANONYMOUS}, use {\ttfamily Authenticated\+Voter\+::\+IS\+\_\+\+AUTHENTICATED\+\_\+\+FULLY} or {\ttfamily Authenticated\+Voter\+::\+IS\+\_\+\+AUTHENTICATED} instead.
\item Deprecate {\ttfamily Authentication\+Trust\+Resolver\+Interface\+::is\+Anonymous()} and the {\ttfamily is\+\_\+anonymous()} expression function as anonymous no longer exists in version 6, use the {\ttfamily is\+Full\+Fledged()} or the new {\ttfamily is\+Authenticated()} instead if you want to check if the request is (fully) authenticated.
\item Deprecate the {\ttfamily \$authentication\+Manager} argument of the {\ttfamily Authorization\+Checker} constructor
\item Deprecate setting the {\ttfamily \$always\+Authenticate} argument to {\ttfamily true} and not setting the {\ttfamily \$exception\+On\+No\+Token} argument to {\ttfamily false} of {\ttfamily Authorization\+Checker}
\item Deprecate methods {\ttfamily Token\+Interface\+::is\+Authenticated()} and {\ttfamily set\+Authenticated}, return null from \char`\"{}get\+User()\char`\"{} instead when a token is not authenticated
\item Add {\ttfamily Access\+Decision\+Strategy\+Interface} to allow custom access decision strategies
\item Add access decision strategies {\ttfamily Affirmative\+Strategy}, {\ttfamily Consensus\+Strategy}, {\ttfamily Priority\+Strategy}, {\ttfamily Unanimous\+Strategy}
\item Deprecate passing the strategy as string to {\ttfamily Access\+Decision\+Manager}, pass an instance of {\ttfamily Access\+Decision\+Strategy\+Interface} instead
\item Flag {\ttfamily Access\+Decision\+Manager} as {\ttfamily @final}
