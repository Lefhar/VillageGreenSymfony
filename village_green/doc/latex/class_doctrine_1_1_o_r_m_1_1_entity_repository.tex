\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_entity_repository}{Entity\+Repository}} serves as a repository for entities with generic as well as business specific methods for retrieving entities.

This class is designed for inheritance and users can subclass this class to write their own repositories with business-\/specific methods to locate entities.
