\doxysubsection{Detailed Description}
An \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_event_subscriber}{Event\+Subscriber}} knows himself what events he is interested in. If an \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_event_subscriber}{Event\+Subscriber}} is added to an \mbox{\hyperlink{class_doctrine_1_1_common_1_1_event_manager}{Event\+Manager}}, the manager invokes \mbox{\hyperlink{interface_doctrine_1_1_common_1_1_event_subscriber_a77d372ed259d7e188f408e350d4f1d66}{get\+Subscribed\+Events}} and registers the subscriber as a listener for all returned events.
