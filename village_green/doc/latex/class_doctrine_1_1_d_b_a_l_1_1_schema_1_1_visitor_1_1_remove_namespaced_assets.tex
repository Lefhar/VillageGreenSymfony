\doxysubsection{Detailed Description}
Removes assets from a schema that are not in the default namespace.

Some databases such as My\+SQL support cross databases joins, but don\textquotesingle{}t allow to call DDLs to a database from another connected database. Before a schema is serialized into SQL this visitor can cleanup schemas with non default namespaces.

This visitor filters all these non-\/default namespaced tables and sequences and removes them from the SChema instance.
