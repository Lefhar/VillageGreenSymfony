\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_doctrine_1_1_d_b_a_l_1_1_query_1_1_query_builder}{Query\+Builder}} class is responsible to dynamically create SQL queries.

Important\+: Verify that every feature you use will work with your database vendor. SQL \mbox{\hyperlink{class_doctrine_1_1_d_b_a_l_1_1_query}{Query}} Builder does not attempt to validate the generated SQL at all.

The query builder does no validation whatsoever if certain features even work with the underlying database vendor. Limit queries and joins are NOT applied to UPDATE and DELETE statements even if some vendors such as My\+SQL support it.
