\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_schema_dumper}{Schema\+Dumper}} class is responsible for dumping the current state of your database schema to a migration. This is to be used in conjunction with the \mbox{\hyperlink{class_doctrine_1_1_migrations_1_1_rollup}{Rollup}} class.
