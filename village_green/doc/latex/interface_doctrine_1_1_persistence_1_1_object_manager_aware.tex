\doxysubsection{Detailed Description}
Makes a Persistent Objects aware of its own object-\/manager.

Using this interface the managing object manager and class metadata instances are injected into the persistent object after construction. This allows you to implement Active\+Record functionality on top of the persistence-\/ignorance that Doctrine propagates.

Word of Warning\+: This is a very powerful hook to change how you can work with your domain models. Using this hook will break the Single Responsibility Principle inside your Domain Objects and increase the coupling of database and objects.

Every \mbox{\hyperlink{interface_doctrine_1_1_persistence_1_1_object_manager}{Object\+Manager}} has to implement this functionality itself. \begin{Desc}
