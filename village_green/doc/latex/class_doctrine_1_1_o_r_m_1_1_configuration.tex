\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_doctrine_1_1_o_r_m_1_1_configuration}{Configuration}} container for all configuration options of Doctrine. It combines all configuration options from DBAL \& ORM.

Internal note\+: When adding a new configuration option just write a getter/setter pair.
