Checkout a socket from the specified URI authority.

The resulting socket resource should be checked back in via {\ttfamily \mbox{\hyperlink{interface_amp_1_1_socket_1_1_socket_pool_a63e2e5ab53c42f724e3a5ff8a52fd8b8}{Socket\+Pool\+::checkin()}}} once the calling code is finished with the stream (even if the socket has been closed). \mbox{\hyperlink{class_amp_1_1_failure}{Failure}} to checkin sockets will result in memory leaks and socket queue blockage. Instead of checking the socket in again, it can also be cleared to prevent re-\/use.
