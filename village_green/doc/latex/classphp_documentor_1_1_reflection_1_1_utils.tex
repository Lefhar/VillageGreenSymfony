Wrapper function for phps preg\+\_\+split

This function is inspired by \mbox{\hyperlink{}{https\+://github.\+com/thecodingmachine/safe/blob/master/generated/pcre.\+php}}. But since this library is all about performance we decided to strip everything we don\textquotesingle{}t need. Reducing the amount of files that have to be loaded, ect.

\hline
string & {\em \$subject} & The input string. \\
\hline
int & {\em \$limit} & If specified, then only substrings up to limit are returned with the rest of the string being placed in the last substring. A limit of -\/1 or 0 means \char`\"{}no limit\char`\"{}. \\
\hline
int & {\em \$flags} & flags can be any combination of the following flags (combined with the $\vert$ bitwise operator)\+: {\itshape PREG\+\_\+\+SPLIT\+\_\+\+NO\+\_\+\+EMPTY} If this flag is set, only non-\/empty pieces will be returned by preg\+\_\+split(). {\itshape PREG\+\_\+\+SPLIT\+\_\+\+DELIM\+\_\+\+CAPTURE} If this flag is set, parenthesized expression in the delimiter pattern will be captured and returned as well. {\itshape PREG\+\_\+\+SPLIT\+\_\+\+OFFSET\+\_\+\+CAPTURE} If this flag is set, for every occurring match the appendant string offset will also be returned. Note that this changes the return value in an array where every element is an array consisting of the matched string at offset 0 and its string offset into subject at offset 1.\\
