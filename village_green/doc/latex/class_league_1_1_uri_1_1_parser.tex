Extract components from an URI containing a colon.

Depending on the colon \char`\"{}\+:\char`\"{} position and on the string composition before the presence of the colon, the URI will be considered to have an scheme or not.


\begin{DoxyItemize}
\item In case no valid scheme is found according to RFC3986 the URI will be parsed as an URI without a scheme and an authority 
\item In case an authority part is detected the URI specific part is parsed as an URI without scheme 
\end{DoxyItemize}

ex\+: email\+:\href{mailto:johndoe@thephpleague.com}{\texttt{ johndoe@thephpleague.\+com}}?subject=Hellow\%20World!

This method returns an associative array containing all the URI components.

{\footnotesize\ttfamily parse (\begin{DoxyParamCaption}\item[{string}]{\$uri }\end{DoxyParamCaption})}

Parse an URI string into its components.

This method parses a URI and returns an associative array containing any of the various components of the URI that are present.

{\ttfamily  \$components = (new Parser())-\/\texorpdfstring{$>$}{>}parse(\textquotesingle{}\href{http://foo@test.example.com:42?query\#'}{\texttt{ http\+://foo@test.\+example.\+com\+:42?query\#\textquotesingle{}}}); var\+\_\+export(\$components); //will display array( \textquotesingle{}scheme\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}http\textquotesingle{}, // the URI scheme component \textquotesingle{}user\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}foo\textquotesingle{}, // the URI user component \textquotesingle{}pass\textquotesingle{} =\texorpdfstring{$>$}{>} null, // the URI pass component \textquotesingle{}host\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}test.\+example.\+com\textquotesingle{}, // the URI host component \textquotesingle{}port\textquotesingle{} =\texorpdfstring{$>$}{>} 42, // the URI port component \textquotesingle{}path\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}\textquotesingle{}, // the URI path component \textquotesingle{}query\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}query\textquotesingle{}, // the URI query component \textquotesingle{}fragment\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}\textquotesingle{}, // the URI fragment component ); }

The returned array is similar to PHP\textquotesingle{}s parse\+\_\+url return value with the following differences\+:


\begin{DoxyItemize}
\item All components are always present in the returned array 
\item Empty and undefined component are treated differently. And empty component is set to the empty string while an undefined component is set to the {\ttfamily null} value. 
\item The path component is never undefined 
\item The method parses the URI following the RFC3986 rules but you are still required to validate the returned components against its related scheme specific rules. 
\end{DoxyItemize}
