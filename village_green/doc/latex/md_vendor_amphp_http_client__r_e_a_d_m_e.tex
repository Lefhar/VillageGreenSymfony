\doxysection*{}

This package provides an asynchronous HTTP client for PHP based on \href{https://github.com/amphp/amp}{\texttt{ Amp}}. Its API simplifies standards-\/compliant HTTP resource traversal and RESTful web service consumption without obscuring the underlying protocol. The library manually implements HTTP over TCP sockets; as such it has no dependency on {\ttfamily ext/curl}.\hypertarget{md_vendor_amphp_http_client__r_e_a_d_m_e_autotoc_md35}{}\doxysubsection{Features}\label{md_vendor_amphp_http_client__r_e_a_d_m_e_autotoc_md35}

\begin{DoxyItemize}
\item Supports HTTP/1 and HTTP/2
\item \href{examples/concurrency/1-concurrent-fetch.php}{\texttt{ Requests concurrently by default}}
\item \href{examples/pooling/1-connection-count.php}{\texttt{ Pools persistent connections (keep-\/alive @ HTTP/1.\+1, multiplexing @ HTTP/2)}}
\item \href{https://amphp.org/http-client/follow-redirects}{\texttt{ Transparently follows redirects}}
\item \href{examples/basic/7-gzip.php}{\texttt{ Decodes compressed entity bodies (gzip, deflate)}}
\item \href{examples/basic/1-get-request.php}{\texttt{ Exposes headers and message data}}
\item \href{examples/streaming/1-large-response.php}{\texttt{ Streams entity bodies for memory management with large transfers}}
\item \href{https://amphp.org/http-client/requests\#request-method}{\texttt{ Supports all standard and custom HTTP method verbs}}
\item \href{examples/basic/4-forms.php}{\texttt{ Simplifies HTTP form submissions}}
\item \href{examples/basic/1-get-request.php}{\texttt{ Implements secure-\/by-\/default TLS ({\ttfamily https\+://})}}
\item \href{https://github.com/amphp/http-client-cookies}{\texttt{ Supports cookies and sessions}}
\item \href{https://github.com/amphp/http-tunnel}{\texttt{ Functions seamlessly behind HTTP proxies}}
\end{DoxyItemize}\hypertarget{md_vendor_amphp_http_client__r_e_a_d_m_e_autotoc_md36}{}\doxysubsection{Installation}\label{md_vendor_amphp_http_client__r_e_a_d_m_e_autotoc_md36}
This package can be installed as a \href{https://getcomposer.org/}{\texttt{ Composer}} dependency.


\begin{DoxyCode}{0}
\DoxyCodeLine{composer require amphp/http-\/client}

\end{DoxyCode}
