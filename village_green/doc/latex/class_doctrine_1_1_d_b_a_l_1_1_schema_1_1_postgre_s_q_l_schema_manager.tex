{\footnotesize\ttfamily get\+Schema\+Search\+Paths (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

The search path for namespaces in the currently connected database.

The first entry is usually the default namespace in the \mbox{\hyperlink{class_doctrine_1_1_d_b_a_l_1_1_schema_1_1_schema}{Schema}}. All further namespaces contain tables/sequences which can also be addressed with a short, not full-\/qualified name.

For databases that don\textquotesingle{}t support subschema/namespaces this method returns the name of the currently connected database.
