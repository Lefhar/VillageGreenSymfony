Processes a row of the result set.

Used for identity-\/based hydration (HYDRATE\+\_\+\+OBJECT and HYDRATE\+\_\+\+ARRAY). Puts the elements of a result row into a new array, grouped by the dql alias they belong to. The column names in the result set are mapped to their field names during this procedure as well as any necessary conversions on the values applied. Scalar values are kept in a specific key \textquotesingle{}scalars\textquotesingle{}.
