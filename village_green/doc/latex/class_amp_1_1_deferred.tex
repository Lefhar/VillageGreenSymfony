\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_amp_1_1_deferred}{Deferred}} is a container for a promise that is resolved using the \mbox{\hyperlink{class_amp_1_1_deferred_a5103de34e145f0a92b4bcfa53c379d58}{resolve()}} and \mbox{\hyperlink{class_amp_1_1_deferred_a5df97dc03c3c97cdf40b90fbf101f54f}{fail()}} methods of this object. The contained promise may be accessed using the \mbox{\hyperlink{class_amp_1_1_deferred_a544e3978ada76faea2f7c4f5ef4d438b}{promise()}} method. This object should not be part of a public API, but used internally to create and resolve a promise.
